C Programming Keywords

Keyword Description (English) Description (Roman Urdu) Example
auto Declares automatic variables. Automatic variables declare karta hai. auto int x;
break Exits a loop or switch. Loop ya switch se exit karta hai. break;
case Defines a branch in a switch statement. Switch statement mein aik branch define karta hai. case 1: ...
char Declares a character variable. Character variable declare karta hai. char c = 'A';
const Declares constant variables. Constant variables declare karta hai. const int y = 10;
continue Skips the current iteration of a loop. Loop ki current iteration skip karta hai. continue;
default Specifies the default block in a switch. Switch mein default block specify karta hai. default: ...
do Starts a do-while loop. Do-while loop start karta hai. do { ... } while(condition);
double Declares a double precision floating-point variable. Double precision floating-point variable declare karta hai. double z = 0.1234;
else Specifies the alternative block in an if statement. If statement mein alternative block specify karta hai. else { ... }
enum Defines an enumeration. Aik enumeration define karta hai. enum colors {RED, GREEN, BLUE};
extern Declares an external variable or function. External variable ya function declare karta hai. extern int count;
float Declares a floating-point variable. Floating-point variable declare karta hai. float temperature = 98.6;
for Starts a for loop. For loop start karta hai. for (int i = 0; i < 10; i++) { ... }
goto Transfers control to a labeled statement. Control ko aik labeled statement par transfer karta hai. goto label;
if Starts an if statement. If statement start karta hai. if (condition) { ... }
int Declares an integer variable. Integer variable declare karta hai. int age = 20;
long Declares a long integer variable. Long integer variable declare karta hai. long distance = 1000;
register Declares a register variable. Register variable declare karta hai. register int count = 0;
return Exits a function and optionally returns a value. Function se exit karta hai aur optionally aik value return karta hai. return 0;
short Declares a short integer variable. Short integer variable declare karta hai. short num = 10;
signed Declares a signed variable. Signed variable declare karta hai. signed int value = -1;
sizeof Returns the size of a data type or variable. Data type ya variable ka size return karta hai. sizeof(int);
static Declares a static variable. Static variable declare karta hai. static int counter = 0;
struct Defines a structure. Aik structure define karta hai. struct Person { ... };
switch Starts a switch statement. Switch statement start karta hai. switch (expression) { ... }
typedef Creates a new data type name. Aik naya data type name create karta hai. typedef int myInt;
union Defines a union. Aik union define karta hai. union Data { ... };
unsigned Declares an unsigned variable. Unsigned variable declare karta hai. unsigned int count = 5;
void Specifies that a function returns no value. Specify karta hai ke function koi value return nahi karta. void myFunction() { ... }
volatile Indicates that a variable may be modified by external factors. Indicates karta hai ke variable external factors se modify ho sakta hai. volatile int flag = 1;
while Starts a while loop. While loop start karta hai. while (condition) { ... }
